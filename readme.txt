Sharelock4Wp widget shows the recent posts on Sharelock.io

== Description ==
Sharelock4Wp adds a widget to your site showing the recent posts on Sharelock either for all or just one specific user.

== Installation ==

1. Upload the plugin to your blog or extract the ZIP to the `wp-content/plugins` folder.
2. Activate it 
3. Add the widget to your pages from `Appearance->Widgets` menu by dragging it to the sidebar.
4. You can set displayed number of recent posts and the username you want to follow.
